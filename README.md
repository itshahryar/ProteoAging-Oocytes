## Overview

**ProteoAging-Oocytes** is a bioinformatics web platform designed for **students and researchers** to analyze **gene expression**, visualize **Differentially Expressed Genes (DEGs)**, and explore structured **biological data tables** in an interactive, user-friendly way.  

It helps transform raw gene/proteomics data into **insightful visualizations and tables** without coding.
